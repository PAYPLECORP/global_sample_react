## Documentation

📂 **.env** 계정 관리 파일
#### 해외결제 연동
>📂 **/order &nbsp;:** &nbsp;상품 주문<br>
>📂 **/order_confirm &nbsp;:** &nbsp;주문확정 및 결제<br>
>📂 **/order_result &nbsp;:** &nbsp;결제결과<br>
>📂 **/order_billingKey &nbsp;:** &nbsp;빌링키 결제<br>
#### REST API
>📂 **/payBillkey &nbsp;:** &nbsp;빌링키 결제<br>
>📂 **/cancel &nbsp;:** &nbsp;결제취소<br>
